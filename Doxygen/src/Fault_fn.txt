/**

\fn ARM_FaultClear
Clear the fault buffer.

\fn ARM_FaultOccurred
Call during startup to check if a fault occurred that can be sent, e.g. using \ref ARM_FaultPrint or \ref ARM_FaultRecord.
Consider calling \ref ARM_FaultClear after sending the fault buffer to prevent resending.

\fn ARM_FaultSave
Call from a fault handler to save current fault context into fault buffer.
Consider calling \ref ARM_FaultExit to trigger a processor reset after saving the fault context.

\fn ARM_FaultExit
Call after saving the fault context (see \ref ARM_FaultSave) to trigger a processor reset for recover device from fault state.

\fn ARM_FaultPrint
Decode stored fault context from fault buffer to standard out via a custom routine. To use \ref ARM_FaultPrint one needs
to add the ARM_FaultPrint template implementation to the project.

*/
