/**

\fn ARM_FaultClear
Clear the fault buffer.

\fn ARM_FaultOccurred
Call during startup to check if a fault occurred that can be sent, e.g. using \ref ARM_FaultPrint or \ref ARM_FaultRecord.
Consider calling \ref ARM_FaultClear after sending the fault buffer to prevent resending.

\fn ARM_FaultSave
Call from a fault handler to save current fault information into fault buffer.

\fn ARM_FaultExit
Callback function called after saving the fault information (see \ref ARM_FaultSave).
The default implementation will do a System Reset.
If System Reset is not desired please provide a custom implementation of this function.

\fn ARM_FaultPrint
Decode stored fault information from fault buffer to standard out via a custom routine. To use \ref ARM_FaultPrint one needs
to add the ARM_FaultPrint template implementation to the project.

*/
